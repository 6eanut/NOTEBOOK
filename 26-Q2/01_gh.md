# gh Startup

## startup

```
gh auth login

# 去这里拿token https://github.com/settings/tokens

gh auth status

gh auth logout
```

## 一台机器 多个gh token

```
mkdir -p ~/.config/gh-1
GH_CONFIG_DIR=~/.config/gh-1 gh auth login
mkdir -p ~/.config/gh-2
GH_CONFIG_DIR=~/.config/gh-2 gh auth login

# 在不同的shell下用
export GH_CONFIG_DIR=~/.config/gh-1
# 或
export GH_CONFIG_DIR=~/.config/gh-2
```
